## Задача

Приложение было сделано по просьбе актёра озвучания для одного его сайд-проекта. Он пишет цельную аудидорожку с разными репликами в ней. **Задача** — автоматизировать разбиение цельной аудиодорожки на отдельные файлы.

## Решение

Проект представляет из себя портативное настольное приложение под OS Windows, которое использует возможности FFMpeg для работы с аудиофайлами для поиска фрагментов с низким уровнем шума и вырезания более громких частей в отдельные файлы
