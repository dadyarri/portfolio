## Задача

Разработка способа расширить документ Word своими действиями, в том числе кастомными формами

## Решение

Был написан шаблон для MS Word и набор расширяющих скриптов, которые добавляют форму на WPF (.NET Framework 4.8). Если создать документ из этого шаблона, то в нём появится новая вкладка с собственными действиями. Например, в рамках этого проекта была создана форма, которая позволяет автоматически заполнить договор
